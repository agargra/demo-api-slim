# Slim API Rest Demo

Demo of API Rest using Slim PHP Framework


## Running app


1. Download from git

```
$> git clone https://github.com/agargra/demo-api-slim.git
```


2. Install packages
 
```
$> cd demo-api-slim
```

```
$> composer install
```


3. Configure env vars

```
$> cp .env.example .env
```


4. Edit .env and set vars



5. Init database

```
$> composer database
```


6. Start app

```
$> composer start
```



## Testing

```
$> composer test
```
